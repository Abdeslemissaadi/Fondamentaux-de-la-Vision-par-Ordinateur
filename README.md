# Vision par Ordinateur : Convolution et Filtrage d'Images

## Description
Ce projet explore les techniques fondamentales de traitement d'image numérique. L'objectif est de reconstruire manuellement les opérations mathématiques de base qui sous-tendent les systèmes de vision par ordinateur modernes (IA, reconnaissance d'objets).

## Concepts implémentés
- **Extension de dynamique** : Normalisation du contraste (étalement de l'histogramme).
- **Filtres Passe-Bas** : Application de noyaux moyenneurs pour le lissage et le réduction de bruit.
- **Détection de contours (Gradient)** : Implémentation du filtre de Sobel pour l'extraction de caractéristiques géométriques (gradient en X et Y).

## Compétences Démontrées
- **Mathématiques appliquées** : Compréhension du fonctionnement des matrices de convolution.
- **Manipulation d'images** : Conversion RGB vers niveaux de gris, traitement matriciel des pixels.
- **Rigueur algorithmique** : Implémentation "from scratch" sans dépendance excessive aux fonctions de haut niveau.

## Technologies
- **Langage** : Python
- **Librairies** : `numpy` (calcul matriciel), `matplotlib` (visualisation), `cv2` (lecture/écriture d'images).

---
*Ce projet démontre une compréhension approfondie du traitement des données visuelles, une compétence clé pour des applications en robotique, imagerie médicale ou contrôle qualité automatisé.*
